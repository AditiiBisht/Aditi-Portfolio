## ⚙️ Installation

Clone the repository:

```bash
git clone https://github.com/AditiiBisht/Aditi-Portfolio.git
```

Install dependencies:

```bash
npm install
```

Run the project:

```bash
npm start
```

Build for production:

```bash
npm run build
```

Deploy:

```bash
npm run deploy
```
